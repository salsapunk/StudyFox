## **1. Introdução**

O StudyFox é um sistema de gerenciamento de tarefas voltado para o público 
estudante em geral. Ele apresenta uma interface clara e intuitiva para 
facilitar a organização de ciclos de estudos com a definição de prazos e 
ordenação com base nas matérias, tarefas e anotações definidas pelo usuário.

## 2.Problema e o Objetivo do Sistema

A alta customização e complexidade dos sistemas de organização levam alguns
estudantes a desistirem de organizar sua vida acadêmica.
Pensando nisso, o sistema procura resolver esses problemas por meio de uma interface simples,
clara, intuitiva e orientada à matérias.

## **3. Requisitos Funcionais (RF)**

| Código | Requisito |
| --- | --- |
| RF01 | O sistema deve permitir o registro de novos usuários; |
| RF02 | O sistema deve permitir o login de usuários registrados; |
| RF03 | O sistema deve permitir que o usuário crie/defina matérias; |
| RF04 | O sistema deve permitir que o usuário crie tarefas dentro das matérias; |
| RF05 | O sistema deve permitir que o usuário crie anotações dentro das tarefas e matérias; |
| RF06 | O sistema deve permitir que o usuário defina prazos para as tarefas; |
| RF07 | O sistema deve permitir que o usuário edite matérias e tarefas; |
| RF08 | O sistema deve permitir que o usuário possa excluir matérias e tarefas; |
| RF09 | O sistema deve permitir que o usuário customize o tema do sistema. |

## **4. Requisitos Não Funcionais (RNF)**

| Código | Classificação | Requisito |
| --- | --- | --- |
| RNF01 | Segurança | Apenas usuários autenticados podem acessar as matérias e tarefas; |
| RNF02 | Desempenho | A criação/edição de matérias e tarefas devem demorar no máximo 1 segundo; |
| RNF03 | Disponibilidade | O sistema deve estar disponível a maior parte do tempo; |
| RNF04 | Escalabilidade | O sistema deve suportar o aumento no número de matérias e tarefas, além do número de usuários; |
| RNF05 | Manutenibilidade | O sistema deve permirtir alterações sem mudanças em todas as partes; |
| RNF06 | Usabilidade | O usuário deve conseguir, facilmente, criar, editar e excluir matérias e tarefas em poucos passos. |
